## Getting Started

### For Development
```bash
git clone https://github.com/lightuptw/ai-kanban.git && cd ai-kanban
cd frontend && npm install && cd ..
cd backend && cargo build && cd ..
# Terminal 1: cd backend && cargo run --bin kanban-backend
# Terminal 2: cd frontend && npm run dev
# Terminal 3: opencode serve --port 4096
```
